# Soz Senkron

Tarayicida calisan basit bir soz-senkron aracidir. Ses dosyanizi yukleyin, sozleri satir satir yapistirin, vokal girislerinde isaretleyin ve YouTube uyumlu SRT dosyasi indirin.

## Ozellikler

- Ses veya video dosyasi yukleme
- Sozleri satirlara ayirma
- Her satir icin zaman isaretleme
- SRT olusturma, indirme ve kopyalama
- Proje indir/yukle
- Tarayicida gecici kayit

## Kullanim

`index.html` dosyasini tarayicida acmaniz yeterlidir.
